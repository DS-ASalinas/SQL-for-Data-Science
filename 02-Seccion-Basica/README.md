# Bloque 02: Sección Básica

## Objetivo
Implementar mecanismos de integridad referencial mediante identificadores únicos y dominar el filtrado avanzado de datos para la extracción de información estratégica.

## Conceptos Aplicados
* **Identificadores (PK/FK):** Definición de Primary Keys y Foreign Keys para garantizar la integridad de los datos.
* **Modelado Relacional:** Interpretación de diagramas Entidad-Relación y navegación en esquemas complejos (Base de datos Northwind).
* **Filtrado Crítico:** Uso de cláusulas `WHERE`, operadores lógicos (`AND`, `OR`, `NOT`) y operadores de comparación.
* **Búsqueda de Patrones:** Aplicación de `LIKE` con comodines y gestión de rangos con `BETWEEN`.
* **Tratamiento de Nulos:** Identificación y manejo de registros incompletos mediante `IS NULL` e `IS NOT NULL`.
* **Optimización de Salida:** Ordenamiento con `ORDER BY`, uso de `ALIAS` para legibilidad y control de volumen con `LIMIT`.
