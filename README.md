# SQL for Data Science

Este repositorio documenta mi proceso de especialización en SQL, enfocado en la integridad de datos, optimización de consultas y análisis estadístico. 

## Estructura del Proyecto

El contenido está organizado por niveles de complejidad, siguiendo un esquema de aprendizaje incremental:

* **01-Primeros-Pasos:** DDL y DML básico. Creación de tablas y manipulación inicial.
* **02-Seccion-Basica:** Relaciones, llaves primarias/foráneas y filtrado avanzado.
* **03-Seccion-Intermedia:** Funciones de agregación, Joins y Subconsultas.
* **04-Seccion-Avanzada:** Transacciones e integración con Python.

## Tecnologías
- **Motor:** SQLite3
- **Herramientas:** DB Browser for SQLite
- **Integración:** Python (Pandas, Matplotlib)
