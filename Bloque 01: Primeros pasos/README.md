# Bloque 01: Primeros Pasos

## Objetivo
Establecer las bases de la manipulación de datos (DML) y definición de datos (DDL) mediante la creación de estructuras simples y la inserción de registros.

## Conceptos Aplicados
- **CREATE TABLE**: Definición de tipos de datos (`INTEGER`, `TEXT`, `REAL`).
- **INSERT INTO**: Carga de registros individuales y múltiples.
- **SELECT**: Proyección de columnas específicas.

## Reto Resuelto
**Escenario:** Gestión de inventario básico para una tienda de mascotas.
**Solución:** Se implementó una tabla `Mascotas` para trackear nombres, especies, edades y precios, asegurando el uso correcto de tipos de datos numéricos para transacciones financieras (precios).
